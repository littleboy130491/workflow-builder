# Step 04: Reference Identification — Inputs

## Upstream Dependencies

- `outputs/01-url-intake/reference-urls.txt` — User-provided reference URLs (if any)

## Required Inputs

- User-provided reference URLs (if "NONE", AI must find references)
- If no user references exist, AI researches best-in-class reference sites

## Output Location

- `outputs/04-reference-sites/references.md` — Identified reference sites organized by perspective

## Reference Perspectives

1. **Design**: Visual design, layout, typography, color, imagery
2. **Content**: Copy, messaging, structure, tone
3. **User Flow**: Navigation, CTAs, forms, conversion patterns
4. **Technical**: Performance, accessibility, SEO best practices

Note: Multiple references per perspective are acceptable and encouraged.
