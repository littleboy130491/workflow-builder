# Step 01: URL Intake — Inputs

## User-Provided Inputs

The user must provide:

- **Website URL**: The full URL of the website to be redesigned (e.g., `https://example.com`)
- **Reference URLs** (optional): One or more reference websites the user wants to emulate or draw inspiration from

## Validation

- Website URL must be a valid, accessible URL
- Reference URLs (if provided) must be valid and accessible

## Output Location

- `outputs/01-url-intake/website-url.txt` — Contains the target website URL
- `outputs/01-url-intake/reference-urls.txt` — Contains any user-provided reference URLs (one per line), or "NONE" if not provided
