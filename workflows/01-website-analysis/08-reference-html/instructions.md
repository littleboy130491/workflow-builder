# Step 08: HTML Extraction (References) — Instructions

## Objective

Extract HTML source from reference sites for pattern analysis.

## Process

1. Read references from `outputs/04-reference-sites/references.md`

2. For each reference site:
   - Fetch HTML from the reference URL
   - Save to appropriate subfolder in `outputs/08-reference-html/`
   - Focus on pages that exemplify the reference perspective

3. Create summary files in `outputs/08-reference-html/summaries/{perspective}/{site-name}.md`:
   - Key structural patterns observed
   - Notable design choices (CSS patterns, class naming conventions)
   - Specific techniques used (animations, layout methods, component structures)

4. Capture 1-2 key pages per reference site

## Notes

- Focus on the HTML/CSS patterns that make the reference effective
- Note any framework-specific patterns (Bootstrap, Tailwind, custom CSS)
- Pay attention to accessibility attributes and semantic HTML usage
- Look for reusable component patterns
