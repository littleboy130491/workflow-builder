# Step 06: Screenshot Capture (References) — Inputs

## Upstream Dependencies

- `outputs/04-reference-sites/references.md` — Identified reference sites by perspective

## Required Inputs

- List of reference URLs from `outputs/04-reference-sites/references.md`

## Output Location

- `outputs/06-reference-screenshots/` — Contains screenshots from reference sites
  - Subfolder per perspective: `design/`, `content/`, `user-flow/`, `technical/`
  - Filename format: `{site-name}-{specific-page}.png`

## Screenshot Requirements

- Capture relevant pages from each reference site
- Focus on sections that exemplify the perspective (e.g., hero section for design, CTA area for user flow)
- Full-page or section-specific as appropriate
