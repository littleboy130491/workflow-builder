# Step 09: Page Analysis (Existing Site) — Inputs

## Upstream Dependencies

- `outputs/05-screenshots/` — Screenshots of core pages
- `outputs/07-html-extraction/` — HTML source of core pages

## Required Inputs

- Screenshots and HTML for each core page

## Output Location

- `outputs/09-page-analysis/{page-name}.md` — Analysis for each core page
- `outputs/09-page-analysis/summary.md` — Consolidated summary of all findings

## Analysis Framework

For each page, identify:

### Strengths
- What's working well
- Good design patterns
- Effective content or messaging
- Successful user flows

### Weaknesses
- Usability issues
- Outdated design elements
- Poor navigation
- Slow loading elements
- Accessibility barriers

### Gaps
- Missing features
- Content gaps
- Unaddressed user needs
- Accessibility issues
- Conversion obstacles
