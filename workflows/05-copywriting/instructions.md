# 05 Copywriting

## Brief Description

Write production-ready site copy for each planned page using the approved content direction and messaging framework.

## Expected Input

- `outputs/03-content-direction/site-map.md`
- `outputs/03-content-direction/recommended-direction.md`
- `outputs/04-messaging-framework/messaging-framework.md`
- `outputs/04-messaging-framework/claim-validation.md`

## Expected Outputs

Create the following files inside `outputs/05-copywriting/`:

- One Markdown file per page, for example `home.md`, `about.md`, `services.md`, `contact.md`
- `seo-metadata.md`: Page titles, meta descriptions, and social sharing copy
- `microcopy.md`: CTA labels, form labels, helper text, and trust-supporting UI copy

## Success Criteria

- Every page in the approved site map has complete copy
- Copy is scannable, specific, and aligned with the audience and positioning
- CTA pathways are clear and intentional
- Metadata and microcopy are consistent with the site narrative

## Constraints

- Avoid filler and generic agency-style language
- Keep claims consistent with `outputs/04-messaging-framework/claim-validation.md`
- Do not change the strategic direction during copywriting without documenting the reason

## Execution Notes

- Use real section structure and section intent, not just prose blocks
- Write with implementation in mind so sections can translate cleanly into components
